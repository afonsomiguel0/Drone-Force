O Problema

Contextualização do projeto.

Incêndios em Portugal

Portugal sofre anualmente com incêndios florestais. A recuperação é lenta porque:

  • Os terrenos são muito inclinados.
  • É perigoso para trabalhadores.
  • O custo da mão de obra é elevado.
Nota: Em 2017 arderam cerca de 500.000 hectares.

A Solução Técnica

Hardware e Software.

O Drone DF-X1

O drone foi desenhado para carregar peso e disparar sementes. Utilizamos modelos hexacopteros para maior estabilidade.

Característica Valor
Autonomia45 Minutos
Carga Útil15kg
Velocidade10 m/s

Bio-Cápsulas

As sementes são encapsuladas num composto especial para aumentar a taxa de sobrevivência em solos queimados.

  • Hidrogel para reter água.
  • Nutrientes iniciais.
  • Proteção contra animais.

Perguntas Frequentes

Sim, tem proteção básica (IP54), mas evitamos voar com chuva forte.

Metodologia

Como funciona o processo.

Fase 1: Mapeamento Aéreo

Recolha de dados topográficos do terreno.

Fase 2: Análise de Dados

Processamento da informação para criar a grid de voo.

Fase 3: Planeamento Algorítmico

O algoritmo define os locais exatos.

Fase 4: Disparo das Sementes

Execução do voo e libertação das cápsulas.

Fase 5: Crescimento

Monitorização da recuperação da floresta.
